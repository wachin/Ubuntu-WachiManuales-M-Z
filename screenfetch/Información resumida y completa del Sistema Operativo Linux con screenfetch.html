Información resumida y completa del Sistema Operativo Linux con screenfetch

Necesitaba saber la información del sistema operativo Linux resumida y  completa para hacer un video para Youtube, y la mejor herramienta es esta:

Instalar:
sudo apt-get install screenfetch
y poner en la terminal:
screenfetch
y aparecerá así:
