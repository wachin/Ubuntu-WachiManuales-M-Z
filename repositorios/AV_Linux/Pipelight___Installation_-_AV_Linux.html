Installation - AV Linux
Quote by AVLinux

AV Linux is a free custom shop modded and rodded 32bit computer Operating System designed to turn a regular old (or fairly new) PC or Intel Mac into an Audio/Graphics/Video workstation appliance. [â€¦]

We provide a repository for AVLinux, so that you will get updates automatically without any further interaction. You simply need to import our repository key and add the repository to your sources. All the following commands must be executed as root:

To add the key simply execute:

wget http://repos.fds-team.de/Release.key
apt-key add Release.key

Now you can add the repository by creating a file /etc/apt/sources.list.d/pipelight.list with the following content:

deb http://repos.fds-team.de/stable/debian/ avlinux main

The next step is to install the pipelight-multi and to update your plugin database.

apt-get update
apt-get install pipelight-multi
pipelight-plugin --update

Some old versions of AVLinux and/or filesystems might not have XATTR enabled by default, which is required to view DRM protected Silverlight content. You can follow this tutorial in the AVLinux forum on how to enable XATTR, if streaming pages do not work properly. For all other non-DRM pages this is not necessary.

Now you everything set up and you can continue by enabling plugins.
