Los siguientes programas se instalaran automáticamente al poner esto en la terminal:

sudo apt-get install gnome-font-viewer florence nautilus gnome-disk-utility dolphin4 qt4-qtconfig ffmpegthumbs kffmpegthumbnailer kde-thumbnailer-openoffice kde-thumbnailer-deb konsole mediainfo-gui puddletag cups-pdf alarm-clock-applet gprename pdfshuffler gnome-terminal goldendict geany rosegarden libreoffice vlc gdebi xfce4-datetime-plugin geeqie soundconverter compiz compizconfig-settings-manager compiz-plugins-extra gnome-exe-thumbnailer convertall keymon kdegraphics-thumbnailers dconf-editor caja engrampa tumbler-plugins-extra caja-open-terminal bleachbit unetbootin usb-creator-gtk converseen hugin photoprint ubuntu-restricted-extras aegisub viewnior gnome-system-monitor breeze-cursor-theme git gedit gedit-plugins pluma clementine meld okular isomaster furiusisomount playonlinux nautilus-dropbox videocut gnome-web-photo


Nota: El paquete playonlinux y nautilus-dropbox al momento de instalarse abren una ventana para confirmar la licencia de ttf-mscorefonts, y dropbox es semejante, y si no se hace esto no sigue la instalación, esto significa que hay que estar atento, por lo cual los pongo aparte:


EXPLICACIÓN DE PARA QUE SIRVEN ALGUNOS DE ESTOS PROGRAMAS:

videocut.- para sacar partes de videos
Discos (gnome-disk-utility).- Sirve para ver todas las particiones y editarlas, se lanza desde la terminal con gnome-disks
Florence.- Teclado virtual
furiusisomount.- Para montar ISO en el nautilus (depende de este)
dolphin4.- Un gran administrador de archivos
qt4-qtconfig.- Sirve para configurar el tamaño de las fuentes de dolphin4 y otras aplicaciones de KDE, este queda instalado en "Administrador de configuración" de Ubuntu Studio, allí abrilo, yo le pongo en la pestaña "Appearance" le pongo en GUI Style "Breeze", y en la pestaña "Fonts" le cambio la que trae por defecto "Sans Serif" por "Noto Sans", "Regular" tamaño 11, y debo de guardar al final
konsole.- La consola para abrir aquí la terminal usando dolphin4
mediainfo-gui.- para ver la información de los videos o audios dandoles clic derecho y abrir con este
puddletag.- Para editar mp3 tags
cups-pdf.- Para imprimir como PDF 
okular.- Para mi el mejor visor de PDF de linux
alarm-clock-applet.- Tiene un temporizador y un despertador, para mi es el mejor
gprename.- Para renombrar archivos
krename.- Para renombrar archivos
pdfshuffler.- Para extraer paginas de un pdf, u otras operaciones
uget.- Para descargar archivos 
goldendict.- Diccionario
rosegarden.- Para editar midi, con QjackCtl
gdebi.- El mejor para instalar DEB
soundconverter + ubuntu-restricted-extras.- Para convertir audios otros formatos, el ultimo paquete instala el codec mp3
compiz.- Lo uso sobre UbuntuStudio en el administrador de ventanas XFCE, es decir pongo en la terminal "compiz --replace" para reemplazarlo, es bueno para hacer tutoriales por el zoom que trae 
gnome-exe-thumbnailer.- Para ver los iconos de los .exe de windows (pues yo uso varios programas de Windows en el Linux instalandolos con Wine los que se pueden o los mas dificilies en PlayOnLinux)
BleachBit.- El CCleaner para Linux(pero su interfaz gráfica no tiene nada que ver, y es más complicado usarlo)
Keymon.- Para ver las teclas que uno aplasta. Sirve cuando uno hace videotutoriales
Geeqie.- Visor de imagenes
kdegraphics-thumbnailers.- Para ver vistas previas de PDF en dolphin4 (pero a esta fecha 2016-07-14 no esta funcionando)
dconf-editor.- Para algunas configuraciones importantes de los administradores de archivos caja y nautilus, y otras cosas del sistema
unetbootin.- Instalar una Distribución Linux a partir de un USB (le quita el spash por defecto de la distribución)
usb-creator-gtk.- Crear un Disco de arranque usando un CD o Imagen de Disco ISO (para GNOME), es mejor queunetbootin (no le quita el spash por defecto de la distribución)
playonlinux.- Para instalar programas de Windows difíciles de hacer funcionar (como e-Sword, Foxit Reader, otros)
caja.- Es un fork de nautilus el administrador de archivos por defecto de ubuntu, para mi es mejor que ese (presente desde Ubuntu 16.04)
engrampa.- Es para caja, sirve para extraer o comprimir archivos
converseen.- Para convertir imagenes en toles (por bastantes)
hugin.- Para hacer panoramas con varias fotos tomadas para tal propósito
aegisub.- Para hacer subtítulos
photoprint.- Para imprimir fotos en albumes
viewnior.- Visor de fotografías e imágenes, como el de Windows
