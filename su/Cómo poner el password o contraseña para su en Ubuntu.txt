

Necesita la contraseña de root para su.

Ubuntu lo establece en caracteres aleatorios para evitar que inicie sesión como root. Esto se debe a que algunos usuarios pueden molestarse por tener que escribir una contraseña cada vez que realizan tareas de administración y deciden iniciar sesión en su escritorio como root, lo cual es MUY mala idea.

Si desea poder usar su o iniciar sesión como root (nuevamente, iniciar sesión en su escritorio como root es una mala idea y nunca es necesario), entonces hay una manera de establecer la contraseña de root a una que conozca. Ingrese este comando y siga las instrucciones:

Código:

sudo passwd

poner la nueva contraseña

Ahora puedes usar su.
