Para instalar desde terminal Todo-en-uno en UbuntuStudio 16.04
▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒
Los siguientes programas se instalaran automáticamente (instalación desatendida) al poner esto en la terminal:

sudo apt-get install gnome-font-viewer speedcrunch florence isomaster gnome-disk-utility furiusisomount dolphin4 konsole furiusisomount mediainfo-gui puddletag cups-pdf alarm-clock-applet gprename krename pdfshuffler uget rar unrar gnome-terminal goldendict geany rosegarden libreoffice python-uniconvertor vlc gdebi xfce4-datetime-plugin geeqie soundconverter compiz compizconfig-settings-manager compiz-plugins-extra gnome-exe-thumbnailer mplayerthumbs convertall keymon kdegraphics-thumbnailers icewm okular dconf-editor caja engrampa caja-open-terminal arandr converseen hugin photoprint playonlinux nautilus nautilus-dropbox clementine kruler calibre mkvtoolnix-gui ppa-purge gwenview gnome-web-photo qt4-qtconfig gedit pluma phototonic gufw

sudo apt-get remove dolphin4 konsole jack-mixer bleachbit psensor

chromium-browser 

El paquete playonlinux nautilus-dropbox ubuntu-restricted-extras al momento de instalarse abre una ventana para confirmar la licencia de ttf-mscorefonts, y dropbox es semejante, y si no se hace esto no sigue la instalación, esto significa que hay que estar atento, por eso los incluyo para después

sudo apt-get install playonlinux nautilus-dropbox


EXPLICACIÓN DE PARA QUE SIRVEN ALGUNOS DE ESTOS PROGRAMAS:

gufw.- Firewall
phototonic.- Utilidad para visualizar imágenes (solo que para ser hagil hay que copiarle la ruta que estemos explorando en algún administrador de archivos "Ctrl + L")
gwenview.- Utilidad para visualizar imágenes
speedcrunch.- Es una calculadora exelente, sólo configure en "Configuración/Carácter separador entero-fracción" si quiere separar con punto o coma, esto porque después no pueda ingresar rápido los valores en su teclado.
ppa-purge.- Sirve para dejar una aplicacion instalada desde PPA a la versión original (revertir lo que se hizo)
mkvtoolnix-gui.- Yo lo utilizo para unir videos en mp4 pues es muy rapido comparandolo con 
Discos (gnome-disk-utility).- Sirve para ver todas las particiones y editarlas, se lanza desde la terminal con gnome-disks
Florence.- Teclado virtual
furiusisomount.- Para montar ISO en el nautilus (depende de este)
dolphin4.- Un gran administrador de archivos
konsole.- La consola para abrir aquí la terminal usando dolphin4
mediainfo-gui.- para ver la información de los videos o audios dandoles clic derecho y abrir con este
puddletag.- Para editar mp3 tags
cups-pdf.- Para imprimir como PDF 
alarm-clock-applet.- Tiene un temporizador y un despertador, para mi es el mejor
gprename.- Para renombrar archivos
krename.- Para renombrar archivos
pdfshuffler.- Para extraer paginas de un pdf, u otras operaciones
uget.- Para descargar archivos 
goldendict.- Diccionario
rosegarden.- Para editar midi, con QjackCtl
gdebi.- El mejor para instalar DEB
soundconverter.- Para convertir audios otros formatos
compiz.- Lo uso sobre UbuntuStudio en el administrador de ventanas XFCE, es decir pongo en la terminal "compiz --replace" para reemplazarlo, este compiz es bueno para hacer tutoriales por el zoom que trae 
gnome-exe-thumbnailer.- Para ver los iconos de los .exe de windows (pues yo uso varios programas de Windows en el Linux instalandolos con Wine los que se pueden o los mas dificilies en PlayOnLinux)
BleachBit.- El CCleaner para Linux(pero su interfaz gráfica no tiene nada que ver, y es más complicado usarlo)
Keymon.- Para ver las teclas que uno aplasta. Sirve cuando uno hace videotutoriales
psensor.- Si lo abro muestra la temperatura de la PC
Geeqie.- Visor de imagenes
kdegraphics-thumbnailers.- Para ver vistas previas de PDF en dolphin4 (pero a esta fecha 2016-07-14 no esta funcionando)
icewm.- El mejor administrador de ventanas para x para PC con bajos recursos(el mejor para mi, es dificil de configurar, pero la PC vuela)
okular.- Es un buen visor de PDF
dconf-editor.- Para algunas configuraciones importantes de los administradores de archivos caja y nautilus, y otras cosas del sistema
nautilus-dropbox.- Es el dropbox pero para integrarlo en el Administrador de Archivos nautilus, se ven los iconos de dropbox allí
mplayerthumbs.- para ver vistas previas de videos dolphin4
playonlinux.- Para instalar programas de Windows difíciles de hacer funcionar (como e-Sword, Foxit Reader, otros)
caja.- Es un fork de nautilus el administrador de archivos por defecto de ubuntu, para mi es mejor que ese (presente desde Ubuntu 16.04)
converseen.- Para convertir imagenes en toles (por bastantes)
hugin.- Para hacer panoramas con varias fotos tomadas para tal propósito
aegisub.- Para hacer subtítulos
photoprint.- Para imprimir fotos en albumes
viewnior.- Visor de fotografías e imágenes, como el de Windows

VirtualBox Reached 5.2 Major Release (How-to Install) | UbuntuHandbook
http://ubuntuhandbook.org/index.php/2017/10/virtualbox-reached-5-2-major-release-how-to-install/
Primero, si usted tenía instalado Virtualbox debe desinstalarlo con estos comandos
sudo apt-get purge virtualbox* && sudo apt-get purge unity-scope-virtualbox
Poner en la terminal una por una las siguientes cuatro lineas de comandos:
sudo sh -c 'echo "deb http://download.virtualbox.org/virtualbox/debian xenial contrib" >> /etc/apt/sources.list.d/virtualbox.list'
wget -q https://www.virtualbox.org/download/oracle_vbox_2016.asc -O- | sudo apt-key add -
wget -q https://www.virtualbox.org/download/oracle_vbox.asc -O- | sudo apt-key add -
sudo apt update && sudo apt-get install virtualbox-5.2
Sobre cómo desinstalar Virtualbox lo vi aquí:
[Solved] VirtualBox fails to start any guest
https://forums.virtualbox.org/viewtopic.php?f=7&t=84990
